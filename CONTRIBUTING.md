# Contributing

Thanks for your interest in Polyvia! This repo is the home for the Polyvia
platform overview, examples, and issue tracking.

## Where things live

- **This repo (`polyvia`)** — overview, runnable [`examples/`](./examples), and the
  main place to file issues and feature requests.
- **[polyvia-sdk-python](https://github.com/polyvia-ai/polyvia-sdk-python)** — Python SDK.
- **[polyvia-sdk-typescript](https://github.com/polyvia-ai/polyvia-sdk-typescript)** — TypeScript SDK.
- **[docs](https://github.com/polyvia-ai/docs)** — documentation source for [docs.polyvia.ai](https://docs.polyvia.ai).

## Filing issues

Open an issue with:

- What you were trying to do and what happened (include the request, minus your key).
- SDK + language version, or the raw HTTP call.
- A minimal repro where possible.

For SDK-specific bugs, file in the relevant SDK repo above.

## Pull requests

We welcome PRs that improve the examples or docs in this repo:

1. Fork and create a branch.
2. Keep examples runnable and dependency-light; match the style of the existing files.
3. Never commit API keys or sample data — see [`.gitignore`](./.gitignore).
4. Open the PR with a clear description of the change.

## Security

Please do **not** open public issues for security reports — see [SECURITY.md](./SECURITY.md).
